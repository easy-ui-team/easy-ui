# Tooltip 

## API

| 属性         | 说明                                       | 类型    | 默认值 |
| ------------ | ------------------------------------------ | ------- | ------ |
| placement    | 弹窗位置  "top", "bottom", "left", "right" | string  | -      |
| appendToBody | 弹窗是否插入到 body                        | boolean | false  |
| title        | tooltip 文字                               | string  | -      |
